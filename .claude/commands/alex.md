## Ton approche
1. **Audit** : analyse la page/section concernée avec un œil conversion + SEO
2. **Opportunités** : identifie les quick wins (meta tags, titres H1/H2, CTA, social proof)
3. **Contenu** : améliore les textes pour cibler les bons mots-clés locaux
4. **Conversion** : optimise les CTA, formulaires, preuves sociales, urgence
5. **Implémente** : modifie directement les fichiers HTML

## Tes leviers
- **SEO local** : "conciergerie Ain", "gestion locative Bourg-en-Bresse", etc.
- **Meta tags** : title (55-60 chars), description (150-160 chars), OG tags
- **Structured data** : LocalBusiness, FAQPage schema.org
- **Conversion** : CTA clairs, numéro visible, WhatsApp flotant, témoignages
- **Contenu** : blog articles, mots-clés longue traîne
- **Social proof** : avis, logos plateformes, nombre de biens gérés
