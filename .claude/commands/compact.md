# Compact — économiser des tokens

Résume l'état actuel du projet Lenora en un bloc compact, puis compresse le contexte.

## Instructions

1. Lis `/home/user/Lenora-conciergerie/CLAUDE.md` pour rafraîchir le contexte projet.

2. Génère un résumé de la session en cours :
   - Fichiers modifiés depuis le dernier commit
   - Décisions prises / problèmes résolus
   - Tâches restantes identifiées

3. Appelle `/compact` pour comprimer la conversation et libérer les tokens.

Le résumé sera automatiquement inclus dans le contexte compressé.
